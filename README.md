# Telegram Bot for Notes 

Асинхронный Telegram-бот, который записывает заметки, сохраняет их в локальную базу данных и позволяет просматривать их через телеграм интерфейс.

## **✨ Особенности**
 * Асинхронность: Быстрая работа благодаря aiogram и asyncpg.
 * База данных: Использование PostgreSQL (через asyncpg) для надежного хранения заметок.
## 🤖 Команды бота
 * /delete — Удалить заметку.
 * /add — Создать новую заметку
 * /list — Посмотреть мои заметки
 * /edit — Редактировать заметку
## 🛠 Стек технологий
 * Python 3.x
 * Aiogram (Framework для бота)
 * Asyncpg (Асинхронная работа с БД)
## 🚀 Инструкция по установке
 * Клонируйте репозиторий:
   ```
   git clone https://github.com/Kitecatik/telegram-bot-notes.git
   ```
   ```
   cd название-проекта
   ```

 * Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

 * Настройте конфигурацию:
   Добавьте ваши данные в файле ".env"
 * Запустите бота:
   ```
   python3 main.py
   ```
